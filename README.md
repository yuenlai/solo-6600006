# solo-6600006 - Cross-platform File Sync & Version Manager

## Tech Stack
- **Frontend**: React + TypeScript + Ant Design
- **Backend**: Rust + Actix-web + Tokio
- **Database**: SQLite + RocksDB

## Quick Start

### Backend
```bash
cd backend && cargo run
```

### Frontend
```bash
cd frontend && npm install && npm run dev
```
